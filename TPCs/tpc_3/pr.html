Transformações com XSLT

Stylesheets em XSLT para criação de páginas HTML



O XSL ( eXtensible Stylesheet Language) é uma linguagem usada na escrita de stylesheets aplicadas a ficheiros XML. Esta, dispõe de uma linguagem de transformação de ficheiros XML em outros ficheiros de formato diferente, o XSLT ( XSL Transformations), sendo usada neste TPC para a criação de páginas HTML apartir da informação contida nos ficheiros XML.

Usando o XSLT, podemos adicionar e/ou remover elementos e atributos ao ficheiro resultante. Podemos então manipular a informação presente no ficheiro XML e apresentá-la da forma desejada, usufruindo para isso da escrita de templates. Relativamente à procura da informação nos ficheiros XML, o XSLT dispõe do XPath, que é a linguagem usada na navegação ao longo dos elementos e atributos deste tipo de ficheiros.

Assim, aquando do processo de transformação, o XSLT usa o XPath para definir as partes do ficheiro XML às quais devem ser aplicadas o(s) template(s) escritos na stylesheet. Quando há uma correspondência, o XSLT aplica as transformações correspondentes para o ficheiro resultante.

Os resultados deste TPC podem ser vistos durante a leitura deste "abstract", uma vez que a página atual foi gerada apartir de uma stylesheet XSLT, assim como na visualização das páginas relativas a arqueosítios disponibilizadas na secção abaixo.
